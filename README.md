# NetCoreAngular

Basic .NET Core Visual Studio 2017 project with [Angular CLI](https://github.com/angular/angular-cli)

## Dependencies

1. Visual Studio 2017
2. .NET Core 1.1
3. Angular CLI 1.0.0
4. NPM


## Install

1. Clone repository
2. cd NetCoreAngular
3. *npm install*
4. open in Visual Studio 2017
5. Change run from IIS Express to NetCoreAngular
6. Run


## Rebuild Angular app

1. cd NetCoreAngular
2. *ng build*
